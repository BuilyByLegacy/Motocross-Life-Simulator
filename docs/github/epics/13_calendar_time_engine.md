# EPIC: Calendar Time Engine

## What
Plan days, weeks, seasons, school years, birthdays, deadlines, aging, and dated history for Motocross: Chasing the Dream. This document defines intent, boundaries, and issue-planning structure only; it does not implement gameplay.

## Why
This epic exists to turn motocross life into remembered human history rather than a stack of disconnected mechanics. It should follow these project principles:

- Every rider has a story. Chase yours.
- Build memories, not mechanics.
- People before players.
- Memories matter more than rewards.
- Authenticity before convenience.
- The world does not revolve around the player.
- Everything has history.
- Decisions echo.
- Respect the family.
- Design for emotion.
- Leave a legacy.

## Major Systems
- Calendar Time Engine planning model
- Memory and history records
- Relationship and consequence hooks
- Authenticity and emotional tone guidelines

- Calendar Time Engine has documented inputs, outputs, and boundaries.
- Child issues are ready for GitHub planning without gameplay implementation.
- Major decisions support people, memory, authenticity, and legacy.
- Related systems have clear integration questions and dependencies.
